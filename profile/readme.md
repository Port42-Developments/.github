## About Us

Port42 Developments is a software development studio focused on creating modern, reliable, and user-friendly applications. We believe in building software that works seamlessly, looks great, and makes a difference.

## Our Values

- **Quality First**: Every line of code matters. We prioritize clean, maintainable, and well-tested software.
- **User-Centered Design**: Our applications are built with the end-user in mind, ensuring intuitive and delightful experiences.
- **Innovation**: We embrace modern technologies and best practices to deliver cutting-edge solutions.
- **Transparency**: Open communication and clear documentation are at the heart of everything we do.

## Technology Stack

We work with modern web technologies including:

- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Cloudflare Workers, Node.js
- **Infrastructure**: Cloudflare Pages, GitHub Actions
- **Tools**: Vite, ESLint, Prettier
